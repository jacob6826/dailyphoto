# Daily Photo

A modern web application built with Next.js app router to ensure you see all your desktop background photos once before they repeat. Now integrated with Netlify Blobs!

## Getting Started

Because this application uses Netlify Blobs, you should run it locally using the Netlify CLI.

1. Install Netlify CLI: `npm install -g netlify-cli`
2. Run the application: `npx netlify dev`
3. View the app at `http://localhost:8888`
